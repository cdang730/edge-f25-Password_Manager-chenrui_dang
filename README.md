# Password Manager
A secure, command-line password manager that encrypts all stored passwords using a master key derived from your master password. It supports safe saving with backups, import/export, and password masking. The program includes lock/unlock functionality, confirmation for destructive actions, and a built-in help menu for easy navigation â€” all while keeping output clean and readable.

## Feature
- User registration and login
- Input validation on master password
- encrypt every password stored
- Add, list, search, edit, and delete passwords
- Import/export password data as JSON
- Automatic backup on save
- Clean terminal interface
